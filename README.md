### Варианты команд для запуска:
- pytest -v --tb=line --language=en 
- pytest -v --tb=line --language=en  -n 3 # запуск тестов в несколько процессов (существенно ускорит прогон, но повышает риск падения тестов)

language - не обязательная опция. По умолчанию "en"

В pages.texts добавлена папка для проверки текста в текущей локали. 
Используется только в "should_not_be_products_in_basket" в целях демонстрации
